# ⚡ Instant Defuse (CS2)

Плагин для **Counter-Strike 2**, который позволяет игрокам стороны Контр-Террористов моментально обезвреживать бомбу при соблюдении определённых условий.

Все условия можно гибко настраивать через конфиг.

---

## 📌 Описание

Плагин добавляет механику **моментального дефуза** для стороны КТ.  
Если игрок пытается начать обезвреживание бомбы и все условия выполнены — бомба разминируется мгновенно.

Если хотя бы одно из включённых условий не соблюдено — моментальный дефуз отменяется, а в чат выводится сообщение с причиной отказа.

---

## ⚙️ Условия

Формат в конфиге:

1 - включить условие  
0 - выключить условие  

По умолчанию включены все условия.

### 1. Достаточно ли времени для обезвреживания
- Учитывается наличие Defuse Kit
- Если времени недостаточно — происходит моментальный взрыв
- ⚠ Это поведение не регулируется конфигом

### 2. Нет ли летящей гранаты, брошенной игроком стороны Террористов

### 3. Не горит ли рядом с бомбой молотов / зажигательная граната

### 4. Есть ли живые игроки команды Террористов

---

## 💬 Сообщения

Если игрок КТ пытается раздефузить бомбу при несоблюдении одного из условий:
- Моментальный дефуз отменяется
- В чат выводится сообщение с причиной отмены

---

## 📦 Установка

1. Установить необходимые зависимости (если требуются)
2. Распаковать архив с плагином в: game/csgo/
3. Настроить конфиг под свои требования
4. Перезапустить сервер

---

## 🔧 Настройка

Все параметры находятся в конфигурационном файле плагина.  
Можно включать или отключать отдельные условия в зависимости от желаемого игрового баланса.
